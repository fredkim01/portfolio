TaskWeb

I created a web application that allows users to create and manage their to-do lists. The website features a clean and intuitive user interface with easy-to-use controls that help users stay organized and productive.

This to-do website help users manage their tasks by providing a simple interface where they can create, edit, and delete tasks as needed. Users can categorize their tasks based on priority, whether it be business or personal.

To create this website, I used HTML5 and CSS3 to design the front-end interface and JavaScript to add interactivity and functionality.
